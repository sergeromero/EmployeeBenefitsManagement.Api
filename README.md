# Employee Management API

Sample ASP.NET Core Web API built using:

- .NET 10
- Clean Architecture
- CQRS
- MediatR
- FluentValidation
- Entity Framework Core
- SQL Server

## Features

- Create Employee
- Get Employee By Id
- Centralized Exception Handling
- Validation Pipeline
